## 1. 디자인 시스템 개요

### 1.1 디자인 철학

푸드케어 클레의 Mobile Web 디자인은 4가지 핵심 가치를 기반으로 합니다:

- **신뢰성 (Reliability)**: 식품 안전과 정기 배송을 담보하는 명확한 정보 제공
- **편의성 (Convenience)**: 직관적인 네비게이션과 빠른 주문 프로세스
- **접근성 (Accessibility)**: 모든 사용자가 쉽게 이용할 수 있는 UI/UX
- **일관성 (Consistency)**: 모든 페이지에서 동일한 디자인 언어 사용

### 1.2 디자인 그리드 시스템
```
뷰포트: 375px × 667px (모바일 웹)
기본 여백: 16px (좌우 padding)
그리드: 8px base grid system
최대 컨텐츠 너비: 343px (375px - 32px padding)
```

### 1.3 해상도별 대응

- **모바일**: 320px ~ 480px (타겟: 375px)
- **태블릿**: 481px ~ 768px (적응형 확장)
- **데스크톱**: 769px+ (웹 버전 별도)

### 7.2 식단 주문 페이지 (Meal Plan Order)

**URL:** `/shop/mealPlan/E/102` (초기1), `/shop/mealPlan/E/101` (준비기)  
**우선순위:** 🔴 1순위 (Critical)

#### 페이지 구조
```
1. 헤더 - 상품명

2. 상품 정보
   - 이미지 (180px, 1:1)
   - 상품명 (20px, Bold)
   - 가격 (18px, Bold, #2B8B60)
   - 별점 + 리뷰 수

3. 3-Step Accordion Form
   - Step 1: 배송정보 (배송 주기, 시작일)
   - Step 2: 주문정보 (옵션, 수량)
   - Step 3: 배송일정 (예정 날짜)

4. 우측 사이드바
   - 배송지 주소
   - 상품 요약
   - 가격 계산
   - 최종 금액

5. 상품 상세
   - 영양정보
   - 고객 리뷰 (리뷰 수)

6. 하단 버튼
   - 다음단계 (Primary)
   - 장바구니 (Secondary)
   - 바로구매 (Secondary)
```

#### Accordion 동작
```
- 한 번에 하나만 열림
- 클릭으로 토글
- 애니메이션: 300ms ease-in-out
```
